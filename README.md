# Kiln-Craft
A responsive multi-page PHP website created for a web development course project. The site showcases artisan members, project galleries, workshops, and a commission request form.

The project was built without JavaScript. Interactive features such as the gallery lightbox and theme switcher are handled with HTML, PHP, and CSS.

## Features

- Reusable PHP header and footer templates
- Multi-page website structure
- Member profiles and project galleries
- Discipline filtering
- CSS-only image lightbox
- CSS-only theme switcher
- Responsive layouts using CSS Grid and Flexbox
- Commission request form
- Server-side form validation
- PHP session-based confirmation page
- Keyboard navigation support
- Accessible labels, focus styles, and ARIA attributes

## Technologies Used

- HTML5
- CSS3
- PHP
- PHP Sessions
- Git
- GitHub
